CodeEditer

一个轻量级的代码编辑器工具，用于快速编写和编辑代码。
CodeEditer 提供简洁直观的编辑环境，适合日常代码编辑、学习编程以及简单开发使用。

该项目旨在打造一个 简单、高效、易用的代码编辑体验。
